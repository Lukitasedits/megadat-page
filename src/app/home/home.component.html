TRABAJOS Y SISTEMAS

Contamos con varios sistemas en el mercado.

SISTEMA DE GESTIÓN

Nuestro sistema de gestión se divide en estos grandes módulos: Ventas, Compras, Productos, Caja, Usuarios Y Libros de IVA . Además se caracteriza por ser facil de usar e intuitivo para el usuario.

SISTEMA VISUAL ELECTRÓNICA

Nuestro sistema Visual Electrónica tiene como objetivo principal facturar electrónicamente de una manera cómoda y sencilla. Es un sistema mucho mas pequeño y económico que el de Gestión. Permite llevar el control de los productos, clientes,proveedores,ventas y compras. Además nos permite sacar el libro de IVA ventas e IVA compras.

Un sistema pequeño y eficaz, totalmente intuitivo que permite:
  1. Autorizar tus facturas de forma rápida y sencilla, retornando un numero de CAE y fecha de vencimiento emitidos por la AFIP.
  2. Exportar tus facturas en formato PDF para guardalas en el equipo, enviarlas por mail o imprimirlas.
  3. Interactuar con tu contador para enviarle los archivos necesarios y facilitar el trabajo de ambas partes.
  4. Para mas información descargue el PDF con todas las especificaciones de nuestro sistema en el siguiente link: Visual electrónica.pdf
  5. Descargá el sistema completo (Con rutina de AFIP para facturar de forma electrónica): Descargar sistema
